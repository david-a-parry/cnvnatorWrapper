This directory contains TQtRootGuiFactory class to provide the default
implementation of the  TGuiFactory class for the Qt-based ROOT applications.
BEGIN_HTML
The ROOT design allows for the different custom  implementations 
of the various ROOT GUI components.

This directory provides the
default ROOT GUI based object factories for Qt-based application.
The non-default pure Qt GUI based implementations are available also from BNL
respository.
<br>
See:
<ul>
<li><a href="ftp://root.cern.ch/root/doc/26ROOTandQt.pdf">
      The Chapter about ROOT/Qt Integration
    </a></li>
<li><a href="http://root.bnl.gov">The full-fledged collection of Qt-based 
        implementation of the other ROOT GUI interfaces.
    </a></li>
<li><a href="http://root.bnl.gov/QtRoot/QtRoot.html#publications">List of
    the Qt and QtRoot related publications and documents </a>
</ul>
END_HTML
