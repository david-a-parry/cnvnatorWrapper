This directory contains the I/O interface classes for the <b>CASTOR</b> system.
This interface uses the rfiod daemon for file transport.
An alternative is to use rootd as file transport daemon (TCastorFile).
For more information about <b>CASTOR</b>, see:
<ul>
<li><a href="http://castor.web.cern.ch/castor/">the CASTOR site</a></li>
<li><a href="http://savannah.cern.ch/files/?group=castor">the CASTOR sources</a></li>
</ul>
